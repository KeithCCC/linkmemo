# ASUKA（LinkMemo）機能サマリー

## 📝 基本ノート機能
- **Markdownエディタ** - 本文はMarkdown記法に対応（Markdown-it使用）
- **複数表示モード** - 編集モード、プレビューモード、横分割、縦分割を切替可能（ショートカットキー対応）
- **自動保存** - 編集内容を自動的に保存
- **タイトル自動生成** - 本文からタイトルを自動抽出
- **ノートのダウンロード** - .txt形式でエクスポート可能

## 🏷️ タグ・整理機能
- **ハッシュタグ自動抽出** - `#tag1`, `＃タグ２`などを自動認識
- **タグフィルタリング** - タグをクリックして絞り込み（include/exclude切替可能）
- **グループタグ** - 複数タグを組み合わせてグループ化（`#group:xxx`形式）
- **タグマイニング** - タイトルと本文から自動的にタグを抽出

## 🔗 Wiki風リンク機能
- **双方向リンク** - `[[ノートタイトル]]`形式でノート間リンク作成
- **バックリンク表示** - 参照元ノートを自動表示
- **リンク候補補完** - `[[`入力時に既存ノートを候補表示
- **存在しないリンク** - 未作成ノートはグレー表示

## 🔍 検索・フィルタリング
- **全文検索** - タイトル・本文・タグを横断検索
- **サイドバー検索** - リアルタイムフィルタリング
- **複数タグ絞り込み** - AND/NOT条件で詳細検索
- **最近使用したノート** - 履歴からクイックアクセス

## 🌐 Chrome拡張機能（Webクリッパー）
- **ワンクリッククリップ** - ツールバーまたは右クリックメニューから即座に保存
- **ChatGPT連携** - ChatGPTの最新応答を自動抽出して保存
- **自動タグ付け** - ソースに応じて`#clipping`、`#chatgpt`タグを自動付与
- **大容量コンテンツ対応** - chrome.storageを使用して長文も対応
- **編集画面に直接遷移** - クリップ後すぐに編集可能

## 🎨 UI・UX機能
- **ダークモード対応** - Light/Dark/Sepiaテーマ切替
- **分割画面リサイズ** - エディタ・プレビューの高さを調整可能
- **スクロール同期** - 横分割モード時にスクロール位置を同期
- **キーボードショートカット** - Ctrl+数字キーで各モード切替
- **レスポンシブデザイン** - スマートフォン・タブレット対応

## 💾 データ管理
- **Supabase連携** - クラウド同期でどこからでもアクセス
- **ローカルストレージ** - 設定やカーソル位置を保存
- **JSON エクスポート** - 全ノートを一括エクスポート
- **JSON インポート** - ノートデータの一括インポート
- **Google認証** - セキュアなユーザー認証

## ⚙️ その他機能
- **カーソル位置記憶** - ノート毎にカーソル位置とスクロール位置を保存
- **フォントサイズ調整** - 小・中・大・特大の4段階
- **ナビゲーション表示切替** - サイドバーの表示/非表示
- **削除確認ダイアログ** - 誤削除防止

## 🔧 技術スタック
- **フロントエンド**: React + Vite
- **スタイリング**: TailwindCSS
- **マークダウン**: Markdown-it + markdown-it-task-lists
- **バックエンド**: Supabase (Auth + Database)
- **拡張機能**: Chrome Extension Manifest V3

## 📋 主な画面
1. **ノート一覧** (`/`) - 全ノート一覧とタグフィルタリング
2. **ノート詳細** (`/note/:id`) - Markdownレンダリング表示
3. **ノート編集** (`/edit/:id`) - リアルタイム編集画面
4. **設定** (`/settings`) - 使い方やヘルプ
5. **拡張機能案内** (`/extension`) - Chrome拡張のセットアップ手順
6. **クリップ** (`/clip`) - 外部からのコンテンツ保存受付

## 🎯 開発コンセプト
ASUKA（LinkMemo）は、Zettelkasten方式のノート取りをベースに、Wiki風のリンク機能とChatGPT連携を組み合わせた、次世代のナレッジマネジメントツールです。思考の断片を自由につなぎ、知識を有機的に成長させることを目指しています。
