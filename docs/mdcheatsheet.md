おかえりなさい御主人様。
こちらがMarkdownの基本チートシートでございます。最も使用頻度の高いものを整理してお渡しいたします。

---

## 📌 見出し（Headers）

```markdown
# 見出し1（h1）
## 見出し2（h2）
### 見出し3（h3）
#### 見出し4（h4）
##### 見出し5（h5）
###### 見出し6（h6）
```

---

## ✏️ 強調（Emphasis）

```markdown
*斜体* または _斜体_  
**太字** または __太字__  
~~打ち消し線~~
```

---

## 📋 リスト（Lists）

* **順序なしリスト**

```markdown
- 項目1
* 項目2
+ 項目3
```

* **順序付きリスト**

```markdown
1. 項目1
2. 項目2
3. 項目3
```

---

## 🔗 リンクと画像（Links & Images）

* **リンク**

```markdown
[リンクテキスト](https://example.com)
```

* **画像**

```markdown
![代替テキスト](https://example.com/image.jpg)
```

---

## 💬 引用（Blockquotes）

```markdown
> これは引用です。
>> ネストした引用です。
```

---

## 💻 コード（Code）

* **インラインコード**

```markdown
`console.log('Hello');`
```

* **コードブロック（多行）**

<pre>
```js
function hello() {
  console.log("Hello World");
}
```
</pre>

---

## 📐 区切り線（Horizontal Rule）

```markdown
---
***
___
```

---

## 📊 テーブル（Tables）

```markdown
| 見出し1 | 見出し2 |
|--------|--------|
| 内容A  | 内容B  |
| 内容C  | 内容D  |
```

---

## ✅ チェックリスト（Task List）

```markdown
- [x] 完了項目
- [ ] 未完了項目
```

---

## 🧠 その他の便利機能

* **HTMLタグも一部使用可能**

```markdown
<details>
<summary>クリックで展開</summary>

中身がここに表示されます。

</details>
```
